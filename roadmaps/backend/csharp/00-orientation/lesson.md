## üìö Core Concepts

### 1. The Terminal (CLI)
The GUI (Graphical User Interface) is a menu for customers. The CLI (Command Line Interface) is the kitchen for chefs.
- `ls` / `dir`: "Show me what's on the table."
- `cd`: "Walk into that room."
- `mkdir`: "Build a new box."

### 2. The Feedback Loop
Coding is a loop:
1.  **Write** (in Editor)
2.  **Run** (in Terminal)
3.  **Read Error** (Analysis)
4.  **Repeat**

### 3. Environment Variables
Secret notes your computer keeps in its pocket. Example: `PATH` tells the computer where to look for tools.

## ‚ö†Ô∏è Common Mistakes
- **Fear of the Black Box:** Avoiding the terminal because "it looks like hacking."
- **Wrong Directory:** Trying to cook dinner while standing in the bathroom. Always check `pwd` (Print Working Directory).
- **Ignoring Errors:** Reading only the last line of an error message instead of the first "Caused by...".
